專案實例

實際解決問題的技術方案

社群平台用戶數據分析平台 SOCIAL INSIGHT

結合爬蟲技術、AI 模型與自動化流程的社群使用者分析系統。系統透過 FastAPI 呼叫後台爬蟲(Playwright)取得 X(Twitter) 的公開資料來源，並透過 n8n 與本地 Ollama 亦或是OpenRouter LLM Models進行使用者特徵分析，最終將結果回傳前端提供即時展示。

Python Fastapi n8n Ollama
X(Twitter) 多模型支持
